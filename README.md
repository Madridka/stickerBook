# Вклейка

«Вклейка» — цифровой альбом футбольных карточек: открывай наборы, собирай коллекцию, подготавливай наклейки и размещай их на страницах чемпионата.

Текущая версия: `1.1.3`. Проект находится в статусе **alpha**: основной игровой цикл уже работает, но содержимое альбома и баланс ещё развиваются.

## Возможности

- полный игровой цикл: получение очков, покупка и открытие наборов, просмотр карточек, подготовка наклеек и заполнение журнала;
- энергия на 100 кликов с непрерывным восстановлением в течение часа и растущей наградой за прогресс заполнения журнала;
- множество случайных мини-игр, позволяющие получить блистер бесплатно раз в настраиваемое время;
- коллекция с фильтрацией, сортировкой, просмотром карточек, мягким удалением и обменом пяти физических повторок на одну из пяти случайных карточек;
- каталог журналов с прогрессом заполнения и интерактивный альбом «Чемпионат мира 2026»;
- данные всех 48 сборных-участниц ЧМ-2026 и полные составы Мексики, Испании, Аргентины, Хорватии и Португалии;
- подготовка наклейки в отдельной мини-игре и свободное размещение карточек с сохранением качества, смещения и угла;
- обычные и особые версии одного игрока могут быть вклеены в общий слот одновременно, а отображаемая в журнале версия выбирается переключателем;
- настраиваемые веса выпадения игроков, включая пониженные шансы для редких карточек;
- локальное сохранение очков, энергии, наборов, оригиналов, повторов и всего прогресса коллекции в IndexedDB.

## Ближайшие шаги

- вынести команды и альбомы в универсальную конфигурацию без привязки к одному журналу;
- добавить страницы остальных сборных, новые карточки и визуальные темы;
- связать общий прогресс коллекции с наградами и новыми наборами.

## Будущие идеи

- обмен карточками;
- достижения и статистика;
- синхронизация коллекции между устройствами.

## Локальный запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```

Правила разработки находятся в [`rules/development.md`](rules/development.md), история версий — в [`src/change-log/CHANGELOG.md`](src/change-log/CHANGELOG.md).
