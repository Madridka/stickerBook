# StickerBook

«Вклейка» — цифровой альбом футбольных стикеров на Vue 3 и TypeScript: от открытия набора до подготовки карточки и её точной вклейки в коллекцию.

Текущая публичная версия: `0.7.7` — редакционный альбом с обложкой, страницами проекта и полноценным разворотом сборной Mexico.

## Текущий MVP

- игровой цикл из кликера, магазина, открытия набора и сохранения пяти выпавших карточек;
- коллекция и отдельное хранилище повторов на Dexie с восстановлением состояния между сессиями;
- интерактивный альбом «Чемпионат мира 2026» с разворотом Mexico на 20 карточек, drag-and-drop, подготовкой наклейки и оценкой качества вклейки.

## Ближайшие шаги

- вынести команды и альбомы в универсальную конфигурацию без привязки к Mexico;
- добавить следующие сборные, их составы, карточки и собственные визуальные темы;
- связать общий прогресс коллекции с наградами и новыми наборами.

## Будущие идеи

- обмен стикерами;
- достижения и статистика;
- синхронизация между устройствами.
