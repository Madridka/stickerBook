# StickerBook

«Вклейка» — цифровой альбом стикеров чемпионата на Vue 3 и TypeScript.

Текущая публичная версия: `0.1.0` — подготовка проекта до MVP.

## Текущий MVP

- базовая навигация между главной, альбомом и магазином;
- каркас коллекции и прогресса;
- подключенные Pinia, TailwindCSS, vue-i18n, Dexie.js и VueUse Motion.

## Ближайшие шаги

- добавить данные команд и страниц;
- реализовать получение наборов и сохранение стикеров в Dexie;
- собрать интерфейс страниц альбома.

## Будущие идеи

- обмен стикерами;
- достижения и статистика;
- синхронизация между устройствами.
