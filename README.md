# Вклейка

«Вклейка» — цифровой альбом футбольных карточек с игровым циклом от получения очков и открытия наборов до подготовки и вклейки наклеек.

Текущая версия: `1.2.0` · статус: **alpha**.

## Возможности

- кликер с энергией на 100 действий, восстановлением в течение часа и бонусом за прогресс альбома;
- магазин наборов и шесть случайных мини-игр для получения бесплатного блистера с дневным лимитом;
- сохранённые сессии открытия наборов по пять карточек без потери наград после перезагрузки;
- Drop Engine V2 с редкостями, настраиваемыми шансами и индивидуальными весами карточек;
- коллекция с поиском, фильтрацией, сортировкой, просмотром, мягким удалением и учётом оригиналов и повторов;
- обмен пяти повторов на одну из пяти случайных карточек с сохранением незавершённого выбора;
- интерактивный альбом ЧМ-2026 с оглавлением, перелистыванием, прогрессом и страницами всех 48 сборных; сейчас доступен демонстрационный режим заполненного альбома;
- трёхэтапная подготовка наклейки и свободная вклейка с сохранением качества, смещения и угла;
- совместное хранение обычных и особых версий игрока в одном слоте с выбором отображаемой карточки;
- типизированный каталог schema v2 для 48 сборных, WebP-подложки всех страниц, полные составы Мексики, Испании, Аргентины, Хорватии и Португалии, а также добавленные карточки Нидерландов и ЮАР;
- локальное сохранение баланса, энергии, наборов, коллекции и прогресса альбома в IndexedDB.
