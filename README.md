# Вклейка

«Вклейка» — цифровой альбом футбольных карточек: открывай наборы, собирай коллекцию, подготавливай наклейки и размещай их на страницах чемпионата.

Текущая версия: `1.0.8`. Проект находится в статусе **alpha**: основной игровой цикл уже работает, но содержимое альбома и баланс ещё развиваются.

## Текущий MVP

- полный игровой цикл из получения очков, покупки и открытия наборов, просмотра карточек и сохранения коллекции;
- интерактивный альбом «Чемпионат мира 2026» с разворотом сборной Мексики, подготовкой наклейки и свободным размещением карточек;
- локальное сохранение очков, наборов, оригиналов, повторов, качества и положения каждой вклеенной карточки.

## Ближайшие шаги

- вынести команды и альбомы в универсальную конфигурацию без привязки к одной сборной;
- добавить новые сборные, составы, карточки и визуальные темы страниц;
- связать общий прогресс коллекции с наградами и новыми наборами.

## Будущие идеи

- обмен карточками;
- достижения и статистика;
- синхронизация коллекции между устройствами.

## Локальный запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```

Правила разработки находятся в [`rules/development.md`](rules/development.md), история версий — в [`src/change-log/CHANGELOG.md`](src/change-log/CHANGELOG.md).
