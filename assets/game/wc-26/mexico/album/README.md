# Mexico album spread

Визуальный разворот коллекции World Cup 2026 / Mexico.

Активные страницы:

- `page-01-cover.png` — одиночная обложка проекта;
- `page-02-info-left.png` — описание проекта из корневого `README.md`;
- `page-03-info-right.png` — последние изменения из `src/change-log/CHANGELOG.md`;
- `page-04-stickers-left.png` — левая страница;
- `page-05-stickers-right.png` — правая страница.

Размер каждой страницы: `1536×1200 px`. PNG содержит только визуальный фон и компактный декор у внешних краёв. Логотип, текст, игроки, карточки, рамки и интерактивная геометрия в изображения не встроены.

Страницы 01–03 используют палитру приложения из `src/assets/main.css`: бумага `#f7f3eb`, чернильный синий `#17212b`, коралл `#e86b52`, мята `#b9d8c2` и золото `#e5b95c`. Это отделяет редакционный раздел проекта от зелёно-красной темы сборной Mexico, сохраняя общую журнальную фактуру страниц 04–05.

На desktop страницы 04–05 образуют единый разворот с сеткой `2 + 4 + 4` на каждой половине. Внутренние края оставлены свободными и соединяются общим журнальным сгибом.

Источником координат является `src/data/wc-26/mexico/album.json`. Компоненты Vue накладывают слоты и карточки поверх PNG независимо от изображения.
