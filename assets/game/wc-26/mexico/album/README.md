# Mexico album pages

Фоновые страницы альбома для коллекции ЧМ-2026, команда Mexico.

Все ассеты имеют размер `390×844 px` и используют палитру глубокого зелёного, красного, кремового и золотого цветов. Геометрические элементы отсылают к футбольной журнальной вёрстке и мексиканским орнаментальным ритмам.

## Страницы

| Файл | Назначение |
| --- | --- |
| `page-01-cover.png` | Обложка с центральной свободной областью под содержимое Vue. |
| `page-02-info-left.png` | Левая информационная страница с пустыми текстовыми блоками. |
| `page-03-info-right.png` | Правая информационная страница, зеркальная композиция. |
| `page-04-stickers-left.png` | Левая страница с пустыми контурами мест для будущих Sticker. |
| `page-05-stickers-right.png` | Правая страница с пустыми контурами мест для будущих Sticker. |
| `page-06-stat-left.png` | Левая статистическая страница с пустой сеткой и слотами данных. |
| `page-07-stat-right.png` | Правая статистическая страница с зеркальной сеткой и слотами данных. |
| `page-08-back.png` | Задняя обложка со свободной центральной областью. |

Страницы не содержат игроков, карточек, наклеек, эмблем и готовых игровых данных. Контуры и пустые панели являются только визуальными местами для будущих компонентов Vue.
