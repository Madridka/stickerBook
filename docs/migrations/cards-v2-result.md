# Результат миграции каталога карточек WC-26 на schema v2

## Схемы данных

- Старая схема: массивы `players.json` с полями `fullName`, `type` и `weight`.
- Новая схема: командные `cards.json` schema v2 с типизированными `kind`, `displayName`, `rarity`, `series`, `finish`, `acquisition` и данными игрока.
- Production runtime загружает каталоги только через централизованный loader и Drop Engine V2.

## Результат

- Мигрировано 48 команд.
- Мигрировано 961 карточка: 960 базовых и 1 special-карточка Spain.
- Для каждой команды сохранено 20 базовых слотов альбома.
- Старые ID и пути изображений перенесены без изменений.
- Legacy loader, Drop Engine V1, legacy-типы и 48 файлов `players.json` удалены.

## IndexedDB

IndexedDB хранит ID карточек, а не объекты каталога. Все прежние ID сохранены, поэтому изменение версии базы данных и преобразование пользовательского прогресса не потребовались.

## Известные ограничения

- В каталоге присутствуют пути к изображениям будущих составов, для которых WebP-файлы ещё не добавлены. Пути перенесены из исходной модели без изменений.
- Миграционные CLI, конфигурация overrides, position cache и отчёты сохранены как история миграции и основа для следующих коллекций.
